8. Virtual Home Lab Setup

This project focuses on building a virtual home lab for cybersecurity and ethical hacking using two virtualization platforms: VMware Fusion and UTM. The lab hosts three operating systems—Ubuntu, Kali Linux, and Windows 11 Pro—allowing for diverse tasks such as networking, penetration testing, and server management.

Ubuntu was installed using VMware Fusion, providing a stable Linux environment for network and server-related tasks. Kali Linux and Windows 11 Pro were installed on UTM. While the installation of Kali Linux was straightforward, Windows 11 Pro encountered a wireless connectivity issue, which was resolved by including the Spice tools package during reinstallation.

This home lab serves as a foundational platform for learning and practicing ethical hacking, penetration testing, and network management, providing a flexible and scalable environment for cybersecurity exploration.
